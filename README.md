# BronVentures - Daniel Bron Professional Portfolio

Public professional portfolio showcasing ventures, achievements, and thought leadership. Optimized for SEO discoverability and Google Knowledge Panel generation.

## Purpose

**Personal Brand Website** for bronventures.com - professional portfolio designed for:
- **Knowledge Panel Strategy**: Structured data for Google entity recognition
- **SEO Optimization**: Schema.org markup, sitemap, meta tags
- **Professional Showcase**: Ventures, speaking engagements, publications
- **Contact Hub**: Centralized professional contact information

## Structure

```
bronventures/
├── prod/           # Production-ready files
│   ├── index.html  # Main portfolio page
│   ├── llms.txt    # LLM-readable bio
│   ├── sitemap.xml # SEO sitemap
│   └── robots.txt  # Search engine directives
├── preview/        # Development iterations
├── assets/         # Logos, images, icons
└── uploads/        # Media files
```
